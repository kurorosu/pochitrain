# 1.2.x Changelog

## v1.2.0 (2026-01-21)

### 概要
- ONNX ツールチェーンと CLI 統合を進め, 推論運用の基盤を整備したリリースです.

### Added
- ONNX モデルエクスポートと推論ツールを追加した ([#84](https://github.com/kurorosu/pochitrain/pull/84), [#90](https://github.com/kurorosu/pochitrain/pull/90)).
  - 学習済みモデルをONNXへ変換し, 推論専用経路で再利用できるようにした.
- ONNX 実行時の GPU 利用可否チェックを追加した ([#93](https://github.com/kurorosu/pochitrain/pull/93)).
  - 実行前にGPU利用可否を判定し, 不要な失敗を減らした.
- CLI と ONNX モジュールのユニットテストを追加した ([#102](https://github.com/kurorosu/pochitrain/pull/102)).
  - 主要分岐をテスト化し, CLI回帰の早期検知を可能にした.

### Changed
- CLI を `pochitrain` モジュールへ統合した ([#95](https://github.com/kurorosu/pochitrain/pull/95), [#103](https://github.com/kurorosu/pochitrain/pull/103)).
  - 実行入口を一本化し, 配布形態と利用手順を単純化した.
- ONNX 関連のロギングを `LoggerManager` ベースに統一した ([#92](https://github.com/kurorosu/pochitrain/pull/92)).
  - ログ出力の形式を統一し, 運用時の可読性を向上させた.
- 設定ロード処理を `ConfigLoader` へ共通化した ([#91](https://github.com/kurorosu/pochitrain/pull/91)).
  - 設定読み込み責務を集約し, 重複コードを削減した.

### Fixed
- なし.

### Removed
- 旧CLIファイルを削除した ([#97](https://github.com/kurorosu/pochitrain/pull/97)).
  - 旧経路を整理し, メンテ対象を現行CLIへ集約した.
