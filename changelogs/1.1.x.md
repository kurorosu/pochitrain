# 1.1.x Changelog

## v1.1.0 (2026-01-18)

### 概要
- Optuna 最適化ワークフローと設定出力の整備を進めた, 最適化機能拡張リリースです.

### Added
- `pochi optimize` サブコマンドを追加した ([#80](https://github.com/kurorosu/pochitrain/pull/80)).
  - CLIからハイパーパラメータ探索を直接実行できるようにした.
- Optuna 最適化結果へパラメータ重要度と統計情報を追加した ([#81](https://github.com/kurorosu/pochitrain/pull/81)).
  - 最適化結果の解釈に必要な補助情報を標準出力へ含めた.
- `optimized_config.py` 出力の設定情報を拡張した ([#77](https://github.com/kurorosu/pochitrain/pull/77), [#79](https://github.com/kurorosu/pochitrain/pull/79)).
  - 再現実験に必要な設定項目を欠落なく保存できるようにした.

### Changed
- 設定ファイル構成を統合し, 型アノテーションと依存関係を整理した ([#73](https://github.com/kurorosu/pochitrain/pull/73), [#79](https://github.com/kurorosu/pochitrain/pull/79)).
  - 設定定義の分散を減らし, 保守性を高めた.
- `ConfigExporter` を SRP 準拠へリファクタリングした ([#82](https://github.com/kurorosu/pochitrain/pull/82)).
  - エクスポート処理の責務を分離し, 拡張しやすい形へ整理した.

### Fixed
- `optimized_config.py` へ `train_transform` と `val_transform` が出力されない問題を修正した ([#76](https://github.com/kurorosu/pochitrain/pull/76)).
  - 最適化後設定だけで学習再現できる状態を保証した.

### Removed
- なし.
